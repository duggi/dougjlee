Advanced Analytics: Massive Dataset Trending

An interactive prototype created to demonstrate possibilities at the edge of data visualization. Try demo!


Individual scores inside a matrix are plotted in height units; a time slider allows the user to view the "landscape" at any particular point in time.

The dimensional plotting allows us to examine the data by volume as well as score and accomodates fluctuating volume naturally. A triptych underneath displays contribution by dimensional subsets of the data.
